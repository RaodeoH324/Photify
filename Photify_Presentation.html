The Vision

Photify was built to solve a critical problem: Music streaming should be about the listener, not the advertiser. While mainstream platforms restrict your freedom and track your data, Photify gives it back. It transforms your personal Google Drive into a premium, high-fidelity, and absolute private streaming ecosystem.

🛠️ The Full Stack

Python 3.x Core engines for shuffling, skipping, and playlist logic.
Node.js Express-based backend for metadata and API services.
React Native Cross-platform mobile UI with Fabric rendering.
Java Native Android modules for background audio control.
SQLite Persistent database for user sessions and library cache.
JSON High-speed metadata structure for song libraries.
Google Drive API Distributed cloud storage for zero-server-cost hosting.
Modern CSS3 Glassmorphism and hardware-accelerated animations.
Expo / EAS Streamlined build system for production APK deployment.
